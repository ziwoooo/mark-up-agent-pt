# Markup Automation Agent - Presentation

Claude Code Skill 기반 마크업 자동 생성 에이전트 발표 자료입니다.

## 핵심 기능

- **다중 프레임워크 지원**: HTML, Nuxt 2/3, React, Next.js 자동 감지
- **CSS 방식 자동 감지**: Tailwind, SCSS, CSS Modules
- **기존 코드베이스 학습**: 컨벤션, 공통 컴포넌트 재사용
- **14가지 품질 검증**: 95점 이상 필수, 자동 수정 지원

## 워크플로우

```
PLAN → LEARN → DESIGN → GENERATE → VALIDATE → OUTPUT
```
